 ### Requirements
   - Animation nodes
   - Blender 2.90 or above
    
 ### How to install?
   - Download the file for appropriate Operating System
   - `Open blender -> Edit -> Preferences -> Add-ons -> Install -> Choose downloaded zip file -> Install Add-on`
